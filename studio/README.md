# Sanity Studio Setup

This folder contains the Sanity Studio configuration for the iComply project.

## Configuration

- **Project ID**: `et4gt387`
- **Dataset**: `production`
- **Base Path**: `/studio`

## Schema

The studio includes a basic blog schema with:
- **Post**: Blog posts with title, slug, author, main image, categories, publish date, excerpt, and body content
- **Author**: Author profiles with name, slug, image, and bio
- **Category**: Post categories with title, slug, and description

## Access

The Sanity Studio is accessible at `http://localhost:8080/studio` when the development server is running.
